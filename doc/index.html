Apt/360 Documentation

Apt360 is an open source Apt processor, based on public-domain source code from the IBM N/C 360 Apt Version IV from the late '60's. It is part of the Aptos cam project.

Apt Manual

The original manual has been painstakingly converted into html form from the original IBM manual. There may, however, be instances where the manual differs slightly from the current program.

Using Apt360

Usage of Apt360 follows the general flow diagram below.

Apt360 consists of a binary file, 'apt360', and two scripts, 'apt' and 'callpost'. The binary simply reads from stdin. To use it, you could use a command such as: 

apt360 <somefile.apt
As a convenience, the script 'apt' allows a command line like: 
apt somefile_1.apt somefile_2.apt
This runs two consecutive files and sets some environment variables, but basically runs the previous command. The "apt" script is the preferred way of running the program. As of 1.0.0, command options are limited to the "--nopost" option, although future versions undoubtedly will have more. The apt360 binary is hardwired to call a bash script named "callpost" which is used to execute a seperate postprocessing program.

Vapt

Apt360 is language based, and it does not come with any means of viewing the geometry you create with it. This is a serious limitation, and Sean Dolan's vapt project is a must-have for doing any serious work in apt. Vapt reads the stdout of apt360, parses it, and displays the geometry in 3-d, with an animated tool-path and the ability to zoom, rotate and scale.

Postp

Postp is a postprocessor developed for use with apt360 and other cam packages. It can be downloaded from the Aptos site as well.

Learning Apt

Apt is a large and powerful language, and giving a detailed treatment of it would require writing a fairly good sized textbook. Rather than try to create a introduction to Apt that would be far too brief and incomplete, I would like to point you in the direction of some good resources for learning Apt.

Probably the best resource I have found is the book Numerical Control Programming in Apt by Irvin H. Kral. It covers Apt in great detail, has numerous examples, and the version of Apt used in the book, Apt-AC, is very similar to Apt360. With the exception of some surface definitions and a few advanced features, the two programs are nearly identical. Most of the examples, with the exception of the polyconic surfaces, should work with minor tweaking, and in fact, many of us have used the examples to verify that apt360 was working properly. This is a good sized textbook, at almost 500 pages. Unfortunately, the book is out of print, although you can get some copies off of Amazon.com and Barnes and Noble.

With the source code is an example directory with several sample programs that you can look at to get an idea of what can be done with the program.

The EMC Documentation Wiki has some good background on Apt360.

Cnc Focus is the home page of a very good Windows based version of Apt. Included on the website is a downloadable pdf file which, although specifically for Personal Apt, covers enough common ground with other Apt processors to be a very good resource. If you are interested in a Windows version of Apt, you might want to download the Visual CamScript program as well.
