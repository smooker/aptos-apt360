Programmer's Notes for Apt360 Source


This documentation is basically an extended series of notes I made while I was
was attempting to understand the source and convert the program to run on a
intel pc. As such, it is not authoritative and undoubtedly contains many errors and
omissions. I'm presenting it simply as a help to whoever might be interested in learning
this program .



ACNTRL - (section 0 ) overall control


ASCNTL - (section 1) language, parse, store canon forms


A2CTRL - (section 2) motion, cut vectors

SECTN3: -(section 3) - copy , transform cut vectors- final cl file

APTPOS: -(section 4) - post process cl file
