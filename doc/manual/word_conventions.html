1.3. Word Conventions

The following rules are used to distinguish the different types of words used throughout this manual when showing a general statement format. (These rules do not apply to actual part programs or sample part program statements in this manual.)

  • All APT vocabulary words are printed in capital letters.

    COOLNT/ OFF

  • All variable symbols start with an S and are printed in capital letters. For example: SPNT is a symbol representing a previously defined point; SLIN is a symbol representing a previously defined line; etc.

  • A word that can be either a symbol for a scalar or an actual numerical value is shown by italicized lowercase letters. For example:

    GOTO/ x, y, z.

  • A word that can be either a symbol representing a geometric object or a nested definition of that object starts with an s and is in lowercase letters. For example: spoint represents either a symbol (SPNT) for point or a nested point definition, and similarly for sline, scircle, etc.

  • Items that are optional are included in brackets.

  • Multiple choices for modifiers that must be included are in braces.

    • Following indicates the feed rate is optional.

      GOTO/ spoint [, feed rate]

    • Following indicates one of the options must be chosen.

  • Following indicates:

    1. One of the forms for the point must be given.

    2. A vector may be specified and, if given, must be one of the two forms.

    3. A feed rate may be specified.

  • Valid APT Examples are highlited as follows:

    P1 = POINT/ 1, 1
    Note

    These examples are for demonstration purposes only. Some of the variables referenced in the examples may not have been defined.
