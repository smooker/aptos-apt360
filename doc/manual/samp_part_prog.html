Sample Part Program

An APT part program consists of statements and data in the language recognized by the APT processor. Using this information, the APT system, in conjunction with a postprocessor, performs the required calculations to produce machine-readable instruction for the numerically controlled machine tool.

Figure 2 illustrates a part to be produced using the APT system. Example 1 is a part program manuscript written to produce this part. The information following Example 1 describes each statement briefly to provide an introduction to the APT system language.

Figure 2. Sample Part Desciption

Example 1. Sample Part Program Manuscript

(1) PARTNO N/C 360 APT SAMPLE PART PROGRAM
(2) SP = POINT/ 0, 0, 0
(3) L1 = LINE/ 4, 0, 0, 4, 8, 0
(4) PT = POINT/ 4.0, 8.0, 0
(5) L2 = LINE/ PT, ATANGL, 45
(6) L3 = LINE/ 8, 12, 0, 12, 12, 0
(7) L4 = LINE/ 14, 5, 0, 14, 10, 0
(8) L5 = LINE/ 0, 2, 0, 10, 2, 0
(9) C1 = CIRCLE/ 12, 10, 0, 2.0
(10) C2 = CIRCLE/ 14, 2, 0, 3.0
(11) INTOL/ 0
(12) OUTTOL/ .005
(13) CUTTER/ .25
(14) SPINDL/ 2000, CLW
(15) COOLNT/ ON
(16) FEDRAT/ 20.0
(17) FROM/ SP
(18) GO/ TO, L1
(19) TLLFT, GOLFT/ L1, PAST, L2
(20) GORGT/ L2, PAST, L3
(21) GORGT/ L3, TANTO, C1
(22) GOFWD/ C1, TANTO, L4
(23) GOFWD/ L4, PAST, C2
(24) GORGT/ C2, PAST, L5
(25) GORGT/ L5, PAST, L1
(26) GOTO/ SP
(27) COOLNT/ OFF
(28) SPINDL/ OFF
(29) FINI

Note

To run the above example with apt360, remove the reference labels (1) through (29), or copy and paste Example 10.1 which is only a slightly modified version of this program.

The following is a statement-by-statement explanation of the information contained in the sample part program manuscript. Frequent reference is made to final cutter locations which are shown in Figure 3.

  1. PARTNO N/C 360 APT SAMPLE PART PROGRAM

    Identifies the part and serves as a title for the program listing, as well as identification.

  2. SP = POINT/ 0, 0, 0

    Assigns the name SP to the point whose coordinates are X=0, Y=0 and Z=0.

  3. L1 = LINE/ 4, 0, 0, 4, 8, 0

    Defines a line through the two points whose coordinates are X=4, Y=0, Z=0 and X=4, Y=8, Z=0.

  4. PT = POINT/ 4.0, 8.0, 0

    Defines the point whose coordinates are X=4, Y=8 and Z=0 (the decimal point is optional).

  5. L2 = LINE/ PT, ATANGL, 45

    Defines the line passing through the point PT making an angle of 45 degrees with the X axis.

  6. L3 = LINE/ 8, 12, 0, 12, 12, 0

    Defines the line passing through the two points whose coordinates are X=8, Y=12, Z=0 and X=12, Y=12 and Z=0.

  7. L4 = LINE/ 14, 5, 0, 14, 10, 0

    Defines the line passing through the points X=14, Y=5, Z=0 and X=14, Y=10 and Z=0.

  8. L5 = LINE/ 0, 2, 0, 10, 2, 0

    Defines the line passing through the points X=0, Y=2, Z=0 and X=10, Y=2 and Z=0.

  9. C1 = CIRCLE/ 12, 10, 0, 2.0

    Defines the circle whose center is at X=12, Y=10, Z=0 and whose radius is 2.

  10. C2 = CIRCLE/ 14, 2, 0, 3.0

    Defines the circle whose center is at X=14, Y=2, Z=0 and whose radius is 3.

  11. INTOL/ 0

    Specifies that the tolerance on the inside of the part is 0. this tolerance represents the maximum amount of gouge allowable.

  12. OUTTOL/ .005

    Specifies that the tolerance on the outside of the part is to be .005. This represents the maximum amount of excess material allowable after machining.

  13. CUTTER/ .25

    Specifies that a cutter is to be used which is .25 in diameter.

  14. SPINDL/ 2000, CLW

    Specifies that the spindle at the machine tool is to be turned on at 2000 rpms, and that the rotation is to be clockwise.

  15. COOLNT/ ON

    Specifies that the coolant is to be turned on.

  16. FEDRAT/ 20.

    Indicates that the feed rate is to be 20.0 inches per minute.

  17. FROM/ SP

    Specifies that the starting reference point for the machining operation is to be the point SP.

  18. GO/ TO, L1

    Specifies that the cutter is to move from its current location (the point SP) until it is just tangent TO the line L1. This would be the point PTA, as shown in Figure 3.

  19. TLLFT, GOLFT/ L1, PAST, L2

    Specifies the following: The cutter is to be on the left of the part (TLLFT). At line L1, the turning direction is to the left (GOLFT). (This direction is always taken as viewed along the current direction of travel.) The cutter is to continue on L1 until it is just PAST the line L2. The final point from this movement would be PTB.

  20. GORGT/ L2, PAST, L3

    Specifies that the turning direction at L2 is to the right (GORGT) and that motion is to continue until the cutter is PAST the surface L3. This would occur at the point PTC.

  21. GORGT/ L3, TANTO, C1

    Specifies that the turning direction at L3 is to the right (GORGT) and that motion is to continue until the cutter is tangent to (TANTO) the circle C1. This would occur at PTD.

  22. GOFWD/ C1, TANTO, L4

    Specifies that the direction to be taken at c1 is forward (GOFWD) and that the motion is to be terminated when the cutter is tangent to (TANTO) the line L4. This would occur at point PTE.

  23. GOFWD/ L4, PAST, C2

    Specifies that the direction at L4 is to be forward (GOFWD) and that motion is to continue until the cutter is PAST the circle C2. This would occur at PTF.

  24. GORGT/ C2, PAST, L5

    Indicates that the turn at C2 is to be to the right and that motion is to continue until the cutter is PAST the line L5. This would occur at PTG.

  25. GORGT/ L5, PAST, L1

    Indicates that the turn at L5 is to be to the right (GORGT) and that motion is to continue until the cutter is PAST line L1. This would occur at PTH.

  26. GOTO/ SP

    Specifies that the cutter is to move directly to the point SP (this would bring the tool back to the original starting point).

  27. COOLNT/ OFF

    Specifies that the coolant is to be turned off.

  28. SPINDL/ OFF

    Specifies that the spindle is to be turned off.

  29. FINI

    Indicates the conclusion of this part program.
