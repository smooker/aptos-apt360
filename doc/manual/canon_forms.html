The various input definitions allowable for geometric surfaces are all transformed by the processor such that, internally, the surface is stored in its canonical form. The actual internal canonical form used by the processor includes, in addition to the geometric canonical form, information denoting surface type, name (if any was assigned), subscript, and reference system for the original definition. The additional data is standard for the various surface types; this chapter defines only the geometric canonical form.

16.1. POINT

Canonical form is (X, Y, Z)

where (X, Y, Z) are the Cartesian coordinates of the points in the given reference system.
