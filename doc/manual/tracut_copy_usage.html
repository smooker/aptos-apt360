13.3. Combined Use of TRACUT and COPY

A COPY loop may be embedded within a TRACUT loop, and a TRACUT loop may be embedded within a COPY loop. An example of each of these capabilities appears below: 

1.  ...
    TRACUT/ SMAT1
    ...
    INDEX/ J
    ...
    COPY/ J, MODIFY, SMAT2, n
    ...
    TRACUT/ NOMORE
    ...
2.  INDEX/ I
    ...
    TRACUT/ SMAT1
    ...
    ...     
    TRACUT/ NOMORE
    ...
    COPY/ I, MODIFY, SMAT2, n

Note

In either case the TRACUT statements redefine the original cutter points only; the TRACUT feature is not accumulative.
