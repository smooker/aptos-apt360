5.7. Optional Feed Rate Specification

As was the case in the FROM and GOTO statements, a feed rate can be specified in any general motion command by adding a scalar variable or number to the end of the statement. For example: 

TLLFT, GORGT/ S1, TANTO, S2, 10.0

The feed rate for this command will be 10.0 inches/minute, and it will be in effect until another feed rate is specified.
