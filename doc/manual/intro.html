Introduction

The most significant modern advance in the manufacturing industry has been the advent of numerical control (N/C). However, this concept of controlling the movements of a machine tool by an input medium, such as punched tape, created a tremendous data handling problem. To solve this problem, several computer programs have been developed to span the gap between the engineering drawing and machine tool control.

This manual describes one such programming system: APT (Automatic Programmed Tool). Figure 1 illustrates how the APT system fits into the manufacturing environment.
