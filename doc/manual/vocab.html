Index

This index includes all the vocabulary words (PARTNO, CIRCLE, CUTTER, GOLFT, etc.) and key part programming phrases (part surface, nested definition, multiple check surface, etc.), together with a cross-reference to the section where they are defined. More than one reference is used if an amplificationof the original definition occirs, or if the item is meaningful only in a specific context.

Certain APT vocabulary words are used to select amongst one or more possible conditions in a part program statement. These words are called "Modifiers" and are denoted as such in the index. Since they are used in a variety of places, no specific cross-reference is given for these words unless special emphasis is required.

Note the vocabulary words were pulled from the source code which is later than the original manual.
