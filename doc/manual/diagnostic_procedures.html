The APT processor provides several aids for locating errors in the part program procedures. The first portion of this chapter discusses the error numbers and associated text that would be printed out in the text of the part program where an error occurred. The last portion of this chapter discusses diagnostic aids that are available to assist in locating errors that are not easily detected.

15.1. Error Messages

A specific error message may result from any of the five basic sections of the APT processor, which include:

  • Section 0: Control Program

  • Section I: Translation and compilation phase

  • Section II: Arithmetic element phase

  • Section III: Edit Phase

  • Section IV: Postprocessor phase

The particular error messages resulting from each section are discussed in the next five sections of the manual. Occasional reference is made to PROTAP or CLTAP codes. This information is described in the section titled "Postprocessor Interface" in the IBM manual N/C 360 APT Application Description (H20-0181-1).

15.1.2. Translation Phase Error Messages
