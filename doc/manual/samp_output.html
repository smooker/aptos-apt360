Chapter 10. Sample Part Program Output

The part programming language required to define the program for the sample part shown in the introduction has been discussed in the preceding chapters. The following pages show the output for that part program (See Sample Part Program). A listing of the input program is given followed by a printout of the cutter location. Two more statements have been included in addition to those shown in the introduction. These are the CLPRNT command (which results in the printout of the cutter location data) and the NOPOST command (which specifies that there is to be no postprocessing).

Example 10.1. Sample Part Program Manuscript (Version 2)

PARTNO N/C 360 APT SAMPLE PART PROGRAM                                  010
CLPRNT                                                                  012
NOPOST                                                                  014
SP = POINT/ 0, 0, 0                                                     020
L1 = LINE/ 4, 0, 0, 4, 8, 0                                             030

L3 = LINE/ 8, 12, 0, 12, 12, 0                                          060

L5 = LINE/ 0, 2, 0, 10, 2, 0                                            080
C1 = CIRCLE/ 12, 10, 0, 2.0                                             090
