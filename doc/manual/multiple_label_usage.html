8.4. Multiple Usage of Statement Labels

The APT processor will indicate an error when the same statement label (ID) is used more than once within the same loop or macro. However, the same symbol used as an ID in one loop or macro can be used as an ID in any other loop or macro within the same part program.

Since IF and JUMPTO may appear only in a loop or macro, and can contain references only to ID's within that same loop or macro, there is no ambiguity in permitting multiple usage of ID's under the specified rules.

It is not recommended that a multiple referenced ID be used with regard to TRANTO or multiple check surface commands. In these cases, the APT processor will transfer to the statement With the last referenced usage of the ID, or, if no such reference exists, it will transfer to the next statement containing the specified ID. A mixture of IF, JUMPTO, multiple check surface and/or TRANTO commands - all utilizing multiple IDs - could be sufficiently complex as to make the end results hard to analyze during part program verification procedures.
