16.14. QADRIC

Canonical form is (A, B, C, D, F, G, H, P, Q, R)

where (A, B, C, D, F, G, H, P, Q, R) are coefficients of the general quadric surface equation:

Ax2 + By2 + Cz2 + D + 2Fyz + 2Gxz + 2Hxy + 2Px + 2Qy + 2Rz = 0
