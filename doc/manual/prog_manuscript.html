Program Manuscript

An English-like language is used to describe the part shown on an engineering drawing. This description takes the form of a sequence of statements defining the part geometry, required cutter operations, and machine tool capabilities. These statements, typed into an editor, become the part programmer's manuscript (or "Part program"). The APT processor converts the APT statements into an intermediate format and performs required calculations. By means of a postprocessor program, the intermediate data is then converted to instructions that can be read by the machine tool controller.
