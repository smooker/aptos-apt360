16.15. TABCYL

Canonical form is:


n, k,
m1, m2, ..., m9,
u1, v1, a1, b1, lth1, max1, min1,
u2, v2, a2, b2, lth2, max2, min2,
...,
un, vn, an, bn, lthn, maxn, minn,
un+1, vn+1

where

  • n

    Total number of points, including the extension points

  • k

    Control word used by section II

  • m1 ... m9

    3x3 rotation matrix containing the direction cosines of the UVW axes in the XYZ coordinate system

  • u1, v1

    Extension point

  • a1, b1

    Coefficients of cubic for first interval

  • lth1

    Length of first interval

  • max1

    Maximum value of cubic for first interval

  • min1

    Minimum value of cubic for first interval

  • u2, v2

    Second point

  • a2, b2

    Coefficients of cubic for second interval

  • lth2

    Length of second interval

  • max2

    Maximum value of cubic for second interval

  • min2

    Minimum value of cubic for second interval

  • ...

  • un, vn

    nth point

  • an, vb

    Coefficients of cubic for nth interval

  • lthn

    Length of nth interval

  • maxn

    Maximum value of cubic for nth interval

  • minn

    Minimum value of cubic for nth interval

  • un+1, vn+1

    Extension point
