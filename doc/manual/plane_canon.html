16.3. PLANE

Canonical form is (A, B, C, D)

where (A, B, C, D) are the coefficients of the equation Ax + By + Cz - D = 0

Note

(A, B, C) are the coefficients of a normalized vector; further, C = 0, since a line is treated as a plane normal to the XY plane.
